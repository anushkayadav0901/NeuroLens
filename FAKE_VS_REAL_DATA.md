### Code That Generated It
```python
# create_sample_nifti.py
brain = np.zeros((64, 64, 64))
# Create sphere
x, y, z = np.ogrid[-32:32, -32:32, -32:32]
mask = x**2 + y**2 + z**2 <= 25**2
brain[mask] = 100 + np.random.randn(*brain[mask].shape) * 10

# Add fake tumor
tumor = np.zeros_like(brain)
tumor[28:36, 28:36, 28:36] = 1  # Small cube
```

## How to Check What You Have

### Method 1: File Size
```bash
# Windows
dir data\BraTS\flair.nii.gz

# If shows ~33 KB → FAKE
# If shows ~7-10 MB → REAL
```

### Method 2: Python Check
```python
import nibabel as nib
import os

flair_path = 'data/BraTS/flair.nii.gz'

# Check file size
size_mb = os.path.getsize(flair_path) / (1024 * 1024)
print(f"File size: {size_mb:.2f} MB")

# Check dimensions
img = nib.load(flair_path)
print(f"Dimensions: {img.shape}")

# Verdict
if img.shape == (64, 64, 64):
    print("❌ FAKE DATA")
elif img.shape == (240, 240, 155):
    print("✅ REAL DATA")
else:
    print("? UNKNOWN")
```
